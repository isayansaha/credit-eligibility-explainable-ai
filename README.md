# Explainable AI Credit Eligibility System

This project predicts loan eligibility using Machine Learning and explains decisions using Google's Gemini API.

## Features
- Loan approval probability
- Risk classification (Low / Medium / High)
- Explainable AI output
- Improvement suggestions

## Tech Stack
- Python
- Scikit-learn
- FastAPI
- Gemini API

## Disclaimer
This system is for educational and advisory purposes only.
